Introduction

Yue is the first Chinese AI assistant that seamlessly integrates with cutting-edge video generation technologies, powered by OpenAI's Sora 2 and Google's Veo 3.1.

Generate stunning, hyperrealistic videos with synchronized audio, dialogue, and sound effects—all from simple text prompts or images. Experience the next generation of AI-powered creativity.

Yue represents a breakthrough in AI-powered video creation, combining the best of both worlds: OpenAI's cutting-edge Sora 2 model and Google's advanced Veo 3.1 technology. Built specifically for the Chinese market, Yue offers natural language processing in Mandarin while delivering studio-quality video outputs.

Why Yue?
Yue (月) means "moon" in Chinese, symbolizing our mission to illuminate the creative process and provide a guiding light for content creators across China.

Getting Started

Installation

Install the Yue SDK using npm or pip:

# Using npm
npm install yue-sdk

# Using pip
pip install yue-sdk

Authentication

Initialize Yue with your API key:

import { YueClient } from 'yue-sdk';

const yue = new YueClient({
  apiKey: 'your-api-key-here',
  model: 'sora-2' // or 'veo-3.1'
});
# Python
from yue_sdk import YueClient

yue = YueClient(
    api_key='your-api-key-here',
    model='sora-2'  # or 'veo-3.1'
)

API Reference

Generate Video

Create a video from a text prompt:

const video = await yue.generate({
  prompt: "一只熊猫在竹林中玩耍，阳光透过树叶洒下来",
  duration: 8,
  resolution: "1080p",
  style: "cinematic",
  audio: true
});

console.log(video.url);

Image to Video

Convert a static image into an animated video:

const video = await yue.imageToVideo({
  image: "./path/to/image.jpg",
  prompt: "使图像中的人物开始说话和移动",
  duration: 5,
  motion: "natural"
});

await video.save("output.mp4");

Video Extension

Extend existing video clips while maintaining consistency:

const extendedVideo = await yue.extend({
  videoPath: "./original.mp4",
  prompt: "继续之前的场景，镜头慢慢拉远",
  additionalDuration: 5
});

Parameters

Available parameters for video generation:

{
  prompt: string,           // Required: Text description
  duration: number,         // 1-8 seconds (Veo) or 1-60 seconds (Sora)
  resolution: string,       // "720p", "1080p", "4k"
  fps: number,             // 24, 30, or 60
  style: string,           // "cinematic", "realistic", "anime", "documentary"
  audio: boolean,          // Enable native audio generation
  model: string,           // "sora-2" or "veo-3.1"
  negativePrompt: string   // What to avoid in the video
}

Models

Sora 2

OpenAI's Sora 2 excels at complex physics simulation and longer video sequences. Best for narrative content, realistic motion, and scenarios requiring accurate object interactions.

Sora 2 Specifications
• Maximum duration: 60 seconds
• Best for: Realistic motion, complex physics
• Audio: Synchronized with video generation
• Processing time: 2-5 minutes per video

Veo 3.1

Google's Veo 3.1 offers faster generation times with enhanced prompt adherence and superior image-to-video capabilities. Ideal for marketing content, product demos, and quick iterations.

Veo 3.1 Specifications
• Maximum duration: 8 seconds
• Best for: Quick generation, image-to-video, style consistency
• Audio: Native audio with rich soundscapes
• Processing time: 30-90 seconds per video

Choosing the Right Model

// Use Sora 2 for complex narratives
if (requiresComplexPhysics || duration > 8) {
  model = "sora-2";
}

// Use Veo 3.1 for quick iterations
if (needsFastGeneration || hasReferenceImage) {
  model = "veo-3.1";
}
